# UI/UX Pro Max - Design Intelligence

Comprehensive design guide for web and mobile applications. Contains UX guidelines, touch interactions, typography pairings, and performance rules.

<Alert type="warning">
**Important:** Accessibility and Touch Interactions are CRITICAL priorities in modern web design.
</Alert>

## Priority Rules

### 1. Accessibility (CRITICAL)
- Minimum 4.5:1 ratio for normal text contrast.
- Visible focus rings on interactive elements.
- Descriptive alt text for meaningful images.
- Tab order matches visual order.

### 2. Touch & Interaction (CRITICAL)
- Minimum 44x44px touch targets.
- Use click/tap for primary interactions (don't rely solely on hover).
- Provide clear error messages near the problem area.
- Add `cursor-pointer` to all clickable elements.

### 3. Performance (HIGH)
- Optimize images using WebP, srcset, and lazy loading.
- Respect `prefers-reduced-motion` for accessibility.
- Reserve space for async content to prevent layout shifts.

### 4. Layout & Responsive (HIGH)
- Minimum 16px body text on mobile for readability.
- Ensure content fits viewport width (no horizontal scrolling).
- Define a clear z-index scale (e.g., 10, 20, 30, 50).
